INTRODUCTION

This work term report covers my experience as a co-op student at Exiger. I spent the previous eight months working at Exiger and had a great experience. This internship took place over two co-op work terms and was from May 2022 - December 2022.

ABOUT EXIGER

Exiger is a global company that specializes in risk and compliance. Exiger is revolutionizing how corporations, government agencies, and banks manage risk and compliance in their third parties, supply chains and customers through its software and tech-enabled solutions. Exiger's mission is to make the world a safer and more transparent place to prosper.

WORK TERM GOALS

My three goals for this work term were to improve my ability to program in Java, learn new development tools commonly used with projects, and learn better coding practices and how large-scale applications are kept organized.

IMPROVE UPON MY ABILITY TO PROGRAM IN JAVA

Improving my ability to program in Java was the goal I focused on most during my work term, and this goal related the most to my job tasks. Java was the only language I used as a software developer Intern at Exiger. As a software developer intern, I knew that proficiency in Java would be essential for my job tasks, and I wanted to ensure that I was comfortable with the language.

LEARN NEW DEVELOPMENT TOOLS THAT ARE COMMONLY USED WITH PROJECTS

The second goal I had going into my work term was to learn new development tools commonly used in projects. I knew that many different tools and technologies were available for software development, and I wanted to ensure that I was familiar with the ones most commonly used in the industry. The different development tools I have learned about and expanded my learning are Docker, Maven, HashiCorp Vault, OpenAPI Specification, Swagger, IntelliJ IDE, and Git. Understanding these new development skills taught me a lot about how to research new technologies effectively.

LEARN BETTER CODING PRACTICES AND HOW LARGE-SCALE APPLICATIONS ARE KEPT ORGANIZED

My final goal for my work term was to learn better coding practices and how large-scale applications are kept organized. I knew that being able to write clean and efficient code was essential for any software developer, and I wanted to make sure that I was following best practices in this regard. I also learned a lot about the development process for creating new features and how the software Jira is used to track progress, which relates to the organization of large-scale applications.


Overall, these goals helped me to focus my efforts and ensure that I was making the most of my work term at Exiger. By the end of my internship, I felt that I had made significant progress toward achieving these goals. I was well-prepared to continue my development as a software developer.

DESCRIPTION OF MY JOB

I spent my eight months at Exiger working on various projects that ranged in size and difficulty. The most interesting project I worked on at Exiger was exploring ways to rotate IP addresses using proxy servers to effectively scrape web data. IP rotation using proxy servers was a new concept for me, and I found it fascinating. The core skills required from me at Exiger I learned in class. However, there were multiple instances where I had to learn new skills on the fly.

PROJECTS I WORKED ON AT EXIGER:

  • Set up automated documentation for 123 API endpoints using Swagger OpenAPI Specification
  • Connected Swagger OpenAPI Specification with a UI to produce an interactive form of documentation
  • Integrated HashiCorp Vault into a project to secure and encrypt start-up credentials
  • Explored ways to rotate IP addresses using proxy servers to effectively scrape web data
  • Worked on converting a project into a spring boot application

RELATION TO ACADEMIC STUDIES

My co-op experience at Exiger developed many new skills that relate to my academic studies. I now have a significantly better understanding of how to code in Java. I can also use Git with much more confidence than before. Finally, I am much better at working with a team, which will aid me in any group projects I have in the future.

CONCLUSION

After completing my work term with Exiger, the skills I increased the most are my ability to program in Java and learning a bunch of small skills that will come in handy as a software developer in the future.
